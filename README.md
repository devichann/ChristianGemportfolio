# Person Search

## Description

Person Search is a Next.js application designed to demonstrate advanced search functionality. It leverages **Next.js 15.1+** and **React 19+** to provide a modern, performant search experience. Users can search for people from a pre-populated list and view detailed information about selected persons.

The application showcases best practices in Next.js development, including Server Components, client-side interactivity, and efficient data fetching patterns.

The upgrade to Next.js 15.1 introduced significant improvements in performance and developer experience, including enhanced support for async components and improved TypeScript integration. The shift to React 19 brings new features like automatic batching and improved rendering performance.

## Features

- **Advanced Search**: Real-time search across a person database
- **Server-Side Rendering**: Optimized performance with Next.js SSR
- **Responsive Design**: Works seamlessly across all devices
- **Type-Safe**: Full TypeScript support for reliability

## Tech Stack

- **Framework**: Next.js 15.1+
- **UI Library**: React 19+
- **Language**: TypeScript
- **Styling**: Tailwind CSS

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm or yarn package manager

### Installation

```bash
npm install
# or
yarn install
```

### Development

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build for Production

```bash
npm run build
npm run start
```
